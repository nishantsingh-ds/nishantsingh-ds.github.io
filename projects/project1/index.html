Project 1: Intelligent Chatbot

Overview

This project is a chatbot designed for hotel reservations, built using RNN, LSTM, BRNN, Encoder-Decoder, and BERT models. It helps users reserve rooms and provides personalized responses based on the user's input.

Key Technologies

  • RNN, LSTM, BRNN
  • BERT, HuggingFace Transformers
  • TensorFlow, Keras
